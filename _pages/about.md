- Completed mechanical arm assembly, servo calibration, and camera binding.
- Collected manipulation data and trained models in a real-world environment.
- Conducted end-to-end evaluation from hardware setup to model deployment.
- Explored the collaboration between visual perception, motion planning, and low-level control.

### Classic CNN Reproduction and Practice

Reproduced several classical computer vision models using PyTorch.

- Implemented AlexNet, VGG, and ResNet from scratch.
- Completed data preprocessing, dataset splitting, training, validation, and inference.
- Improved VGG training stability through Kaiming initialization.

### Scientific Image Object Detection

Built an object-detection workflow for high-noise scientific images.

- Annotated YH2AX images with Label Studio.
- Constructed datasets in YOLO format.
- Trained a lightweight YOLO11 model and evaluated performance with metrics such as accuracy and recall.
